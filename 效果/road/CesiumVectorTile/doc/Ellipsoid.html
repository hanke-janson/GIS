类: Ellipsoid

Ellipsoid

new Ellipsoid(xopt, yopt, zopt)

A quadratic surface defined in Cartesian coordinates by the equation (x / a)^2 + (y / b)^2 + (z / c)^2 = 1. Primarily used by Cesium to represent the shape of planetary bodies. Rather than constructing this object directly, one of the provided constants is normally used.
Parameters:

getSurfaceNormalIntersectionWithZAxis(position, bufferopt, resultopt) → {Cartesian3|undefined}

Computes a point which is the intersection of the surface normal with the z-axis.
Parameters:
Name Type Attributes Default Description
position Cartesian3 the position. must be on the surface of the ellipsoid.
buffer Number <optional>
 0.0 A buffer to subtract from the ellipsoid size when checking if the point is inside the ellipsoid. In earth case, with common earth datums, there is no need for this buffer since the intersection point is always (relatively) very close to the center. In WGS84 datum, intersection point is at max z = +-42841.31151331382 (0.673% of z-axis). Intersection point could be outside the ellipsoid if the ratio of MajorAxis / AxisOfRotation is bigger than the square root of 2

 The cartesian to which to copy the result, or undefined to create and return a new instance.
